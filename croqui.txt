Tab 2 - meu studyka (nome do bixinho)

( ok ) Criar um giff utilizando 3 imagens svg, este giff é o bixinho.
( ok ) Modificar o código para que possam ser utilizados vários aparências de bixinhos.
(   ) Criar uma classe contendo as informacoes do bixinho do usuário. E exibir as informações e barras necessárias de forma adequada.
O bixinho terá barra de fome.
O bixinho terá barra de felicidade.
O bixinho terá sua experiência exibida com 2 casas decimais.
O bixinho terá seu level exibido.
O dinheiro será exibido nesta tela.
A barra de fome diminui com o passar do tempo.
A barra de fome aumenta quando se compra comida.
A barra de felicidade aumenta quando se compra brinquedo ou quando se pratica bons hábitos.
A barra de felicidade diminui como tempo e quando se pratica maus hábitos.
A barra de experiência aumenta junto da barra de felicidade quando esta está acima de 80%.
O level aumenta quando o limite de experiência daquele level é alcançado.
O limite de experiência daquele level deve ser aumentado de acordo com o aumento do level.
Pena de morte: Se o bixinho zerar a barra de fome ou de felicidade, ocorre a pena de morte. Nela, o bixinho perde 2 levels e a experiência é zerada. Se o bixinho voltar para o level 1, o dinheiro retorna ao valor inicial.
No início do jogo, um valor de dinheiro inicial é alocado ao usuário.
O dinheiro aumenta quando bons hábitos são praticados.
